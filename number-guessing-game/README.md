## Version 1 — Basic Single Guess

**What It Does:**

- Generates a random number between 1–100.
- User enters one guess.
- If guess equals the random number → Correct.
- Else → Try Again.

---

## Version 2 — Unlimited Attempts + Hints

**What It Does:**

- Runs in a while True loop.
- User guesses until correct.
- Provides hints:
  - Too high
  - Too low.
- Breaks the loop when user guesses correctly.

---

## Version 3 — Limited Attempts + Remaining Attempts Display

**What It Does:**

- Generates:
  - A random number between 1–100
  - A random number of attempts (1–10).
- User plays with limited attempts.
- Gives hints:
  - Too high
  - Too low.
- Shows remaining attempts after each wrong guess.
- If attempts run out → reveals the correct number.
